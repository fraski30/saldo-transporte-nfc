# Saldo Andalucía Granada

Aplicación Android para la consulta de saldo de tarjetas de transporte mediante NFC.  
Permite consultar el saldo de tarjetas del **Consorcio de Transportes de Andalucía** y tarjetas de **autobuses urbanos de Granada**.

---

## Aviso importante
- La aplicación es de **solo lectura**: no permite recargar tarjetas ni pagar con el móvil.  
- Solo muestra la información accesible en la tarjeta.  
- El uso de la aplicación es responsabilidad del usuario.

---

## Requisitos mínimos
- Teléfono Android con **Android 10 o superior**  
- **NFC** habilitado

---

## Funcionalidades
- Muestra el saldo de la tarjeta en pantalla  
- Permite cambiar el nombre de la tarjeta  
- Historial para guardar un registro de cada tarjeta y su saldo  
- Gráfica para visualizar la evolución del saldo  
- Opción para deshabilitar la vibración al leer la tarjeta  
- Modo de interfaz claro, oscuro o automático  
- Acceso mediante huella dactilar  
- Soporte para tarjetas de autobuses urbanos de Granada (desde versión 1.1.0)

---

## Tarjetas compatibles
- Tarjeta del Consorcio de Transporte de Andalucía  
- Tarjetas de autobuses urbanos de Granada
