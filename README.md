# Saldo Transporte NFC

Aplicación Android para la consulta de saldo de tarjetas de transporte mediante NFC.

La aplicación se limita exclusivamente a la lectura de información de la tarjeta y no realiza ninguna modificación sobre la misma.

---

## Características

- Lectura de tarjetas de transporte mediante NFC
- Consulta de saldo disponible
- Solo lectura (no escribe ni altera la tarjeta)
- Aplicación Android

---

## Tarjetas compatibles

Actualmente, la aplicación permite consultar el saldo de:

- Tarjeta del Consorcio de Transporte de Andalucía

---

## Aviso importante

- Esta es una aplicación no oficial.
- No está afiliada, respaldada ni mantenida por ningún organismo de transporte.
- La aplicación se limita a mostrar la información accesible en la tarjeta.
- El uso de la aplicación es responsabilidad del usuario.
